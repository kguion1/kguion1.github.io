## What is Linux?
Linux is an open-source UNIX-like operating system that can be accessed via a terminal. While the phrase "terminal" will be used in this article, you might also hear terms like "console", "shell", and "command line". The differences will not be covered in this page, but can be found [here](https://www.geeksforgeeks.org/difference-between-terminal-console-shell-and-command-line). 
Linux is used to streamline much of the data analysis process. It can be used to run code, view and edit files, rearrange folders, connecting to the cloud or a high performance cluster, and much more!
For beginner data analyzers, an in depth understanding of Linux is not needed. The main "commands" and usage will be covered below. 

## How to use Linux on your computer
To use Linux on your computer, you need a terminal that is Linux-based. For Mac, you are in luck! The default terminal on Macs uses a Linux operating system. For PC computers, we recommend downloading the Ubuntu terminal via [these instructions](https://ubuntu.com/tutorials/ubuntu-on-windows#1-overview). 

### Understanding Linux as a file system
Normally when we are navigating through files on our computer we use a graphical interface, like "Finder" on Macs, or "File Explorer" on PC comptuers. However, we can do the same thing with Linux, via the terminal. We do this so we can more easily understand where our code is being run and how to access different files to use in our analyses.

#### Important note about Ubuntu (SKIP for MAC)
Ubuntu has a specific filing system that requires and extra first step. If you look at your file explorer on PC, you may notice that your Local Disk is listed as a C drive (C:). Opening the Local Disk, there is a folder called "Users" and in it should be a folder with your User's name that was chosen when setting up the computer. We recommend that there are no spaces or special characters, even apostrophes, in this name. You can modify this via [these instructions](https://www.windowscentral.com/how-change-account-name-windows-10-sign-screen).
